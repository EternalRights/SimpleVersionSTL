# SimpleVersionSTL
# 项目简介
本项目基于c++标准模板库**STL**的简单模拟实现，旨在为开发者提供了一种学习和理解 C++ 底层数据结构和算法的途径。  

# 主要功能
- 数据结构实现
  - set: 底层代码[红黑树](https://github.com/EternalRights/SimpleVersionSTL/blob/main/STL%20prepostion/RedBlackTree.h "红黑树代码实现")  

- 算法支持 
# 技术栈
1.编程语言：**c++**  

2.开发工具：**Visual Studio 2022**
